# weather_visualization_Api

基于 node.js 与 Koa 构建的 web 服务，用于 weather_visualization 项目：[HZhertz/weather_visualization 基于 Vue3，TypeScript 与高德地图的全国天气实况可视化](https://github.com/HZhertz/weather_visualization)

## Project setup

### install

```
// npm
npm install

// pnpm
pnpm install
```

### Run serve

```
// npm
npm run serve

// pnpm
pnpm run serve
```
